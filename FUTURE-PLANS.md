### 3. 지도자 과정 수강생 포털 — 중기 프로젝트 (수 주 단위)
로그인이 진짜 가치 있는 대상은 일반 회원이 아니라 **교육생**. 차별화 무기.

- 예: portal.letspilatesla.com 별도 앱 (현 사이트와 같은 React 스택).

## 기술 방향 (포털/VOD 공통)

- **영상 호스팅/보안**: Vimeo Pro/OTT 또는 Mux — 다운로드 방지, 도메인 제한, 서명된 재생 URL. (유튜브 비공개는 유료 콘텐츠에 부적합)
- **인증/DB**: Supabase (또는 Firebase) — 비밀번호를 직접 보관하지 않는 관리형 인증.
- **결제**: Stripe — 카드 정보를 직접 다루지 않음.
- **앱**: 현 사이트와 동일한 React + Vite + Tailwind 스택, 별도 서브도메인 배포.
- 원칙: **영상·결제·인증 인프라는 직접 만들지 않고 관리형 서비스 조합** (보안·유지비 리스크 회피).

## 만들지 않기로 한 것 (재론 시 참고)

- **수업 예약 시스템 + 일반 회원 로그인 자체 구축**: Mindbody가 이미 예약·회원·결제 처리 중. 직접 구축은 결제/개인정보/노쇼 처리 등 유지 부담 대비 이득 없음. Mindbody 구독료가 부담되는 시점에 단계적 재검토.
- **커스텀 어드민 페이지**: 당분간 구글 시트(Registration)가 어드민 역할. 시트가 불편해지는 시점에 재검토.
